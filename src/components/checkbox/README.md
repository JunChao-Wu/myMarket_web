## **checkbox参数说明**
参数名|类型|默认值|可选值|说明
---|:---:|:---:|:---:|---
value|Number|——|——|接受value作为checkbox的值
checkboxStates|String|normal|接受normal, allChecked, noChecked|该参数用来控制checkbox的选择状态

## **checkbox事件**
事件名|回调参数|说明
---|:---:|---
getCheckboxSetion|checkbox的值|接受checkbox选中的值
deleteSeletion|checkbox的值|接收checkbox取消选中的值
